# Лабораторна робота No5

## Допрацювати головну сторінку сайту відповідно до шаблону Portfolio UI - Web
& Mobile – Figma (сторінка Home). Якщо не хочете змінювати головну сторінку,
можете створити нову та розмістити її в перелік своїх лабораторних робіт.

# АБО

## Самостійно знайти відкритий темплейт на figma і реалізувати сторінку по ньому.
При цьому варіанті обов’язково на сайті повинне бути зафіксоване посилання на
темплейт з допомогою елемента на сторінці через position: fixed.
Також, в такому випадку, краще одразу шукайте темплейт з мобільною версією
сайту, так як цей темплейт буде допрацьовуватися в наступних лабораторних.
